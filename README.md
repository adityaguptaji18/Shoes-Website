# Shoes E-Commerce Website (MERN Stack)

A full-stack e-commerce platform for a retail shoes business built using the MERN stack.

## 🚀 Features
- JWT Authentication & Role-Based Access Control
- Product Management with Filtering
- Cart & Wishlist Functionality
- Order Management System
- Razorpay Payment Integration
- Image Upload using ImageKit

## 🛠 Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- JWT
- Razorpay
- ImageKit
